# Project Planning

Plan projects, break down tasks, and coordinate team efforts.

## What I Do
- Define project scope and requirements
- Break down complex projects into tasks
- Create timelines and milestones
- Prioritize work and manage dependencies
- Track progress and identify blockers

## Example Task
"Create a project plan for building a new feature, including tasks, timeline, and team assignments"
